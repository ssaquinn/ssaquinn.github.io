Branch, Pull, Merge

What is main?

Main is the main branch/document. This is the base document/s that is working towards the project function.

Why create a Branch?

A branch is created to maintain the integrity of the main document. This protects the main document from bugs or lost data when changes are made - as the version of main is kept when the branch was made.
